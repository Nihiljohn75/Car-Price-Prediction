# Car Price Prediction using Machine Learning

## Project Overview

This project predicts the selling price of used cars using Machine Learning techniques.

## Objective

Build a regression model that can estimate a used car's selling price based on features such as:
- Present Price
- Car Age
- Kilometers Driven
- Fuel Type
- Transmission
- Seller Type
- Owner Count

## Models Compared

| Model | R² Score | MAE | RMSE |
|---------|---------|---------|---------|
| Linear Regression | 0.818 | 1.142 | 1.698 |
| Random Forest | 0.922 | 0.636 | 1.115 |

## Final Model

Random Forest Regressor

### Results

- R² Score: 0.922
- MAE: 0.636
- RMSE: 1.115
- Cross Validation Score: 0.813
